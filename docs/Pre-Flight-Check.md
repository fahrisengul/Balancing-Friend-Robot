# 🚀 Donanım Hazırlık Kontrol Listesi

Bu liste, parçalar ulaştığında montaj ve test aşamasına geçebilmemiz için gereken fiziksel hazırlıkları kapsar.

### 🔌 Güç Sistemi (Fiziksel)
- [ ] 4S 18650 Piller yuvaya dizilecek (Kutup yönlerine dikkat!).
- [ ] Multimetre ile pil yuvası çıkış voltajı (14.8V - 16.8V) ölçülecek.
- [ ] BMS kartı bağlantısı yapılacak (Güvenli deşarj için).

### 🧠 Kontrol ve Yazılım (Hazırlık)
- [ ] **ST-LINK/V2:** Bilgisayara takılıp sürücüleri (ST-Link Utility) kurulacak.
- [ ] **STM32 Bluepill:** PC13 LED Blink testi yapılacak.
- [ ] **Ugreen NVMe Kutusu:** Samsung SSD içine yerleştirilecek ve formatlanacak.

### 🏎️ Mekanik (4WD)
- [ ] 4WD Şasi parçaları eksiksiz mi? (Motorlar, tekerlekler, vidalar).
- [ ] Motor kabloları lehimlenecek.
